# spoolman-to-prusaslicer
# Spoolman → PrusaSlicer exporter

Skript pro automatický převod cívek ze Spoolmanu do filamentových profilů PrusaSliceru.

Zdroj dat:
http://homeassistant.local:7912

## PrusaSlicer → Spoolman Import

Use `prusaslicer_to_spoolman.py` to import:
- vendors
- filament materials
directly from PrusaSlicer export file PrusaSlicer_config_bundle.ini.
